# Transcripció
## Català
**Transcripció d'àudios del conjunt de dades Mozilla Common Voice.**

Aquesta aplicació està dissenyada per corregir i completar fitxers del conjunt de dades Mozilla Common Voice.

Disposeu d'una versió per a escritori i d'una versió per a web.

### Instruccions d'ús
El procés és el següent:
- En primer lloc se selecciona l'arxiu del conjunt de dades i el seu idioma.
- A continuació, l'aplicació ens mostrarà el contingut del primer registre del conjunt de dades, concretament, ens mostrarà, a l'àrea de text, la transcripció corresponent al registre i al gènere de la veu de l'àudio associat. Si no hubiese transcripció l'àrea de text es mostrarà buit.
- Seguidament podrem escoltar l'àudio o sol·licitar una transcripció automàtica mitjançant els botons corresponents.
- La transcripció mostrada a l'àrea de text pot ser modificada per aconseguir l'àudio corresponent.
- Una vegada conclou les modificacions de la transcripció haureu de guardar el registre i passar al següent mitjançant el botó corresponent.

## English
**Audio transcription from the Mozilla Common Voice dataset.**

This application is designed to correct and complete files from the Mozilla Common Voice dataset.

It has a desktop version and a web version.

### Instructions for use
The process is as follows:
- First select the data set file and its language.
- Next, the application will show us the content of the first record of the data set, specifically, it will show us, in the text area, the transcription corresponding to said record and the gender of the voice of the associated audio. If there was no transcription, the text area will be empty.
- We will then be able to listen to the audio or request an automatic transcription using the corresponding buttons.
- The transcription shown in the text area can be modified to match the corresponding audio.
- Once the modifications of the transcription have been completed, the register must be saved and passed to the next step by means of the corresponding button.


## Español
**Transcripción de audios del conjunto de datos Mozilla Common Voice.**

Esta aplicación está diseñada para corregir y completar archivos del conjunto de datos Mozilla Common Voice.

Dispone de una versión para escriorio y de una versión para web.

### Instrucciones de uso
El proceso es el siguiente:
- En primer lugar se selecciona el archivo del conjunto de datos y su idioma.
- A continuación, la aplicación nos mostrará el contenido del primer registro del conjunto de datos, concretamente, nos mostrará, en el área de texto, la transcripción correspondiente a dicho registro y el género de la voz del audio asociado. Si no hubiese transcripción el área de texto se mostrará vacía.
- Seguidamente podremos escuchar el audio o solicitar una transcripción automática mediante los botones correspondientes.
- La transcripción mostrada en el área de texto puede ser modificada para que concuerde con el audio correspondiente.
- Una vez concluidas las modificaciones de la transcripción deberá guardarse el registro y pasar al siguiente mediante el botón correspondiente.
